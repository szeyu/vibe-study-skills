# Question Type Patterns Reference

Quick reference for advanced question patterns beyond the basics in SKILL.md.

## Scenario-Based MCQ

```markdown
**Scenario:** [Real-world situation description]

**Q.** Based on the scenario above, what would be the most appropriate action?

A) [Option considering factor X only]
B) [Option considering factor Y only]  
C) [Best balanced option - CORRECT]
D) [Common but inappropriate response]
```

## Sequencing Questions

```markdown
**Q.** Arrange the following steps in the correct order:

1. [ ] Step C
2. [ ] Step A
3. [ ] Step D
4. [ ] Step B

<details>
<summary>Answer</summary>
Correct order: A → B → C → D

[Explanation of why this sequence is correct]
</details>
```

## Assertion-Reason Questions

```markdown
**Q.** 

**Assertion (A):** [Statement about a phenomenon]

**Reason (R):** [Statement explaining the cause]

A) Both A and R are true, and R is the correct explanation of A
B) Both A and R are true, but R is NOT the correct explanation of A
C) A is true but R is false
D) A is false but R is true
E) Both A and R are false
```

## Case Study Questions

```markdown
**Case Study:** [Detailed scenario - 3-5 sentences]

**Q1.** [Question about identifying the problem]
**Q2.** [Question about analysis]
**Q3.** [Question about proposed solution]
**Q4.** [Question about potential outcomes]
```

## Diagram-Based Questions

```markdown
**Q.** Refer to the diagram below:

[Mermaid diagram or description]

What does component X represent?
```
